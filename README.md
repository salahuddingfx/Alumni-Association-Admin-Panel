# Practon Alumni Association Platform — Admin Dashboard

This is the control panel and Content Management System (CMS) for the Alumni platform.

## Features
- Dynamic Recharts donation charts
- Member approval logs
- Event scheduler
- Notice board management with real-time push events
- Offline Smart Banglish-to-Bengali transliteration assistant

## Setup and Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run in development mode:
   ```bash
   npm run dev
   ```
